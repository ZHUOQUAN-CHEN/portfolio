Support Vector Machine

SVM is a Support Vector Machine project for binary classification for tumors diagnosis with results of malignant and benign.

1> Exploratory data analysis.
2> Splitting data to training set and testing set.
3> Train the support Vector classifier: fit trainning data set, prediction of testing data set and evaluas by metrics.
4> Using GridSearchCV to find a best globally optimal combination of parameters (C and gamma) as a new estimator for the final prediction.
