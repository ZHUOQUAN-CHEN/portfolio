BANK SHARES ANALYSIS

Bank shares analysis is a Data Analysis project, which included data exploratory analysis and vdata isualization. The analyzed data from yahoo finance, which respectively are Bank of America, CitiGroup, Goldman Sachs, JPMorgan Chase, Morgan Stanley and Wells Fargo between Jan 1st 2006 and Jan 1st 2016.

1> Analysis:
Discovery 1: 4 bank shares had the worst on Jan 20th 2009, which was the day of Obama's inauguration.
Discovery 2: Using Standard Deviation found that the Morgan Stanley was the riskiest stock of those bank shares in 2015.
2> Data Visualization:
Create a distplot chart to show year returns for each stock.
Create a line plot showing the close price for each bank for the entire index of time.
Create a line plot to analyze the moving averages for these stocks in a curtain year.
