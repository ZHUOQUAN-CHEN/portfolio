ARTIST

Artist is a Data Structure project, which is using hash table and visualized data to create a new image with words. Then using Maven, which a tool of building and managing of java, to downloadand all names and version numbers of the dependency jar in the pom file automatically. Lastly, using KUMO to analyze the words frequency.

1> Using HashMap to calculate the words frequency.
2> Filling those words based on the frequency in a specified shape.
3> Set up color, font size and density follow your own idea.
